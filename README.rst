Stopwatch is a pygame based stopwatch.

You can get the code at https://github.com/alej0varas/stopwatch

Install
=======

pip intall -r requirements.txt

Run
===

./main.py

Usage
=====

+-------+--------------------------------------+
|  Key  |                Action                |
+-------+--------------------------------------+
| Space |              Start/Stop              |
+-------+--------------------------------------+
|   r   |                Reset                 |
+-------+--------------------------------------+
|   f   | Swap between fullscreen and windowed |
+-------+--------------------------------------+
|  Esc  |                 Exit                 |
+-------+--------------------------------------+

That's all for now ;)
